# Job Scheduler

This project is a Job Scheduler & Automation Dashboard.

## Tech Stack
- Node.js
- Express
- Frontend (Next.js / React)

## How to run
1. Install Node.js
2. Go to backend folder
3. Run `npm install`
4. Run `node app.js`
